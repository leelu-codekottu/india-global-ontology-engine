# India Global Ontology Engine

An AI-powered intelligence system designed to analyze global events and their impact on India.

The system collects real-time information from news feeds and global datasets, extracts entities and relationships using large language models, and dynamically constructs knowledge graphs to reason over complex geopolitical and economic scenarios.

## Key Features

- Dynamic On-Demand Knowledge Graph Construction
- Hybrid Retrieval (Semantic + Lexical Search)
- GraphRAG-based reasoning engine
- Real-time information ingestion from RSS feeds
- Interactive graph visualization interface
- Evidence-backed decision insights

## Technology Stack

- Memgraph (Graph Database)
- FastAPI (Backend)
- React + Vite (Frontend)
- Groq LLM / Llama Models
- RSS Feed ingestion pipeline

## Use Cases

- Geopolitical analysis
- Economic forecasting
- Strategic intelligence
- Global event impact assessment
